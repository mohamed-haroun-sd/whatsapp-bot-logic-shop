# بوت واتساب ذكي - WhatsApp Bot Logic Shop

## نظرة عامة

بوت واتساب متقدم يجمع بين الذكاء الاصطناعي والقوائم التفاعلية والترحيب التلقائي. يوفر البوت تجربة مستخدم احترافية وذكية لتلبية احتياجات العملاء.

## الميزات الجديدة

### 1. الذكاء الاصطناعي (AI)
- البوت يستخدم **GPT-4o-mini** للرد على الرسائل غير المعرفة مسبقاً
- يمكن للبوت الإجابة على أي سؤال بشكل طبيعي وذكي
- يتعامل مع الأخطاء بأناقة ويعود إلى رسالة خطأ ودية

### 2. القائمة التفاعلية المتقدمة
عند كتابة "قائمة"، سيظهر للمستخدم أربعة خيارات:
- **ساعات العمل:** عرض ساعات عمل الشركة
- **موقعنا:** عرض موقع الشركة (يمكن إضافة رابط خرائط)
- **تواصل مع الدعم:** معلومات التواصل مع فريق الدعم
- **تحدث مع الذكاء الاصطناعي:** الدخول إلى وضع الدردشة الذكية

### 3. نظام الترحيب التلقائي
- عند تواصل مستخدم جديد، يستقبل رسالة ترحيبية تلقائية
- يتم تسجيل المستخدمين الذين تم ترحيبهم لتجنب الرسائل المكررة

## المتطلبات

- **Node.js** (الإصدار 14 أو أحدث)
- **npm** أو **yarn**
- **مفتاح OpenAI API** (متاح كمتغير بيئة `OPENAI_API_KEY`)

## التثبيت

1. استنساخ المستودع:
```bash
git clone https://github.com/your-username/whatsapp-bot-logic-shop.git
cd whatsapp-bot-logic-shop
```

2. تثبيت التبعيات:
```bash
npm install
```

3. إعداد متغيرات البيئة:
تأكد من أن `OPENAI_API_KEY` معرّف في متغيرات البيئة الخاصة بك.

## الاستخدام

لبدء البوت:
```bash
npm start
```

سيظهر رمز QR في الطرفية. قم بمسح الرمز باستخدام تطبيق واتساب على هاتفك لربط البوت.

## الأوامر المتاحة

| الأمر | الوصف |
|------|-------|
| `زر` | اختبار الزر البسيط |
| `قائمة` | عرض القائمة الرئيسية |
| أي رسالة أخرى | الرد من خلال الذكاء الاصطناعي |
